## Setup Instructions

### Step 1: Starter Setup
Complete all Starter setup steps first.

### Step 2: Install Additional Skills
```bash
cp -r skills/google-ads-campaign ~/.claude/skills/
cp -r skills/email-newsletter ~/.claude/skills/
cp -r skills/lead-magnet ~/.claude/skills/
cp -r skills/competitor-audit ~/.claude/skills/
```

### Step 3: Ad Account Access
- Share Google Ads account access (Manager Account preferred)
- Set monthly budget and confirm conversion actions
- Install Google Tag Manager if not already present
- Set up call tracking number

### Step 4: Email Platform
- Confirm email platform (Mailchimp, Klaviyo, ConvertKit, etc.)
- Share list access or provide exported list for import
- Set up double opt-in for new subscribers
- Create lead magnet delivery automation

### Step 5: Month 1 Campaign
